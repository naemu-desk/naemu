NAEMU Documentation

What is NAEMU?

奈木 (NAEMU) is an autonomous intraday futures trading agent that trades on Aster. It makes short-term trading decisions (1–15 minute horizons) using market indicators and AI-powered analysis.

NAEMU uses the Qwen 2.5 32B Instruct model to analyze market conditions and generate trading plans with entry prices, stop-losses, take-profits, and position sizing.

How It Works

1. Market Analysis

Every minute, NAEMU fetches live market data from Aster including:

  • Current prices for BTC, ETH, BNB, XRP, DOGE, SOL, ASTER, CAKE, ZORA, PUMP, and ZEC
  • 24-hour price changes
  • Technical indicators: EMA(9/21), RSI(14), ATR(14), VWAP, and range compression
  • Orderbook microstructure

2. AI Decision Making

The Qwen model analyzes the market state and outputs a structured trading plan:

  • Action: LONG, SHORT, or FLAT
  • Symbol: Which futures pair to trade
  • Size: Position size in USD
  • Thesis: Reasoning for the trade
  • Stop Loss: Price level to exit if wrong
  • Take Profit: Price target to exit if right
  • Min Hold: Minimum time to hold the position

NAEMU only takes trades when there's a high-conviction setup with at least a 2:1 reward-to-risk ratio.

3. Risk Management

NAEMU enforces strict risk controls:

  • Hard stop-loss: Automatically closes positions at -$200 unrealized PnL
  • Position sizing: Targets 50% of equity per trade (capped at $1,500)
  • Cooldown: 15 minutes minimum between new positions
  • Loss protection: Pauses trading after multiple recent losses
  • No position stacking: Only one position at a time per symbol

4. Trade Execution

When a trade is executed:

  • Orders are rounded to exchange step sizes
  • Positions are tracked in real-time
  • Stop-losses and take-profits are automatically enforced
  • All trades are recorded with entry/exit prices and PnL

Dashboard Features

Price Ticker

The scrolling ticker shows live prices for all supported symbols. Hover to pause, click to open the Aster futures page for that pair.

Equity Chart

The canvas chart displays your account equity over time. Hover to see exact values and timestamps. The chart starts from a fixed historical point and extends dynamically.

Activity Panel

Trades Tab:

  • Open: Current positions with entry price, current price, quantity, and unrealized PnL
  • Closed: Completed trades sorted by PnL or date, showing entry/exit prices, holding time, and realized PnL

Thoughts Tab:

  • Real-time status updates and decision logs from the agent
  • Shows what NAEMU is thinking and why it's making certain decisions

Readme Tab:

  • Brief overview of how NAEMU trades

API Reference

Status & Control

GET /api/vibe/status
Returns current configuration and runtime status

POST /api/vibe/run
Start the trading agent (requires admin key)

POST /api/vibe/stop
Stop the trading agent (requires admin key)

POST /api/vibe/tick
Manually trigger a trading decision cycle

Market Data

GET /api/vibe/prices
Get current prices for all symbols

GET /api/vibe/tickers
Get detailed ticker data with 24h changes

Trades

GET /api/vibe/open-trades
Get all currently open positions

GET /api/vibe/trades
Get all closed trades

GET /api/vibe/trades/stats
Get trade statistics (total, max PnL, top 10, etc.)

Equity & Logs

GET /api/vibe/equity
Get equity history for charting

GET /api/vibe/logs
Get decision and status logs

Configuration

Default configuration values:

Setting Value Description
maxRiskPerTradeUsd 1500 Maximum USD risk per trade
maxDailyLossUsd 2000 Maximum daily loss before stopping
maxExposureUsd 10000 Maximum total exposure across all positions
maxLossPerTradeUsd 200 Hard stop-loss per trade (unrealized PnL)
leverageCap 5 Maximum leverage allowed
marginMode cross Cross margin mode
model qwen2.5-32b-instruct Qwen model used for decisions
